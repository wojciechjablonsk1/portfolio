# Projects

## 6-DOF Robotic Arm

[GitHub Repository](https://github.com/wojciechjablonsk1/robotic-arm-6dof-)

A complete mechatronic prototype developed as my engineering thesis project.

### Overview
Designed and developed a six-degree-of-freedom robotic arm combining mechanical design, embedded programming, electronics, and additive manufacturing.

### Main Contributions
- Designed the complete mechanical structure using **SolidWorks**
- Developed CAD models optimized for **FDM 3D printing**
- Designed and assembled the electronic system
- Implemented robot control software using:
  - ESP32 microcontroller
  - C++
  - Python
- Developed custom C++ libraries following ROS-like software architecture
- Integrated motors, electronics, and control software
- Tested and evaluated the prototype
